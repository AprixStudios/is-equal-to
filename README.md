> Determines if x is equal to y

### 🏠 [Homepage](https://github.com/AprixStudios/is-equal-to#readme)

## Install

```sh
npm install
```

## Run tests

```sh
npm run test
```

## Example

```js
const isEqualTo = require('is-equal-to');

isEqualTo(1,2); // False

isEqualTo("hey", "Hey"); // False 

isEqualTo("Hey!", "Hey!"); // True

isEqualTo(2+2,2*2); // True
```
